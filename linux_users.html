Welcome to

Linux Users Group at Indiana University

Linux Users Group at Indiana University is a club for anyone that is interested in computing and learning more about Linux and Free and Open Source Software! Linux operating systems are becoming increasingly popular with home users, and with most large servers in the world using it, there has never been a better time to learn about Linux.

Members will be able to learn about how to use Linux, work with it at a technical level, and discuss and help each other with using it on computers. Anyone is welcome, no matter your technical and computing skill level, even if you have never heard of Linux!.
